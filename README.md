# Final-Exam

NexPlay — Gaming Universe Landing Page

A modern, responsive gaming platform landing page featuring a sleek dark theme, 
interactive elements, and full form validation.

Features:
- Responsive design (320px - 1920px)
- Burger menu navigation
- Form validation with regex (email, password)
- Show/hide password toggle
- API integration (fetch randomuser.me)
- Header background change on scroll
- Scroll-to-top button
- Cookies notification with localStorage
- Hover effects & smooth transitions
- Cross-browser compatible
